# 🏦 Banking Management System

A full-stack banking management system built using Python, Streamlit, and SQLite.  
The project allows admins to manage customers, accounts, transactions, and loans through an interactive dashboard.

---

## Features

- Secure Admin Login
- Multi-Admin Management
- Customer Management
- Bank Account Creation
- Deposit & Withdraw System
- Loan Tracking & Status Updates
- Dashboard Analytics
- Transaction Visualization
- Customer Search Functionality

---

## Tech Stack

- Python
- Streamlit
- SQLite
- Pandas
- Matplotlib
